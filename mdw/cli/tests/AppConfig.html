Workflow: "AppConfig"


Scenario:

  1. In mdw.yaml, the following appears under application:

    application:
  name: my-mdw
  configs:
    - one.properties
    - two.properties
    - three.yaml

  2. The following files exist in the location pointed to by -Dmdw.config.location.
    one.properties

    one.prop=oneValue

    two.properties

    two.prop=twoValue

    three.yaml

    flat.config: threeFlatValue
structured:
  config: threeStructuredValue

  3. The Log Props activity produces this output:

    one: oneValue
two: twoValue
three.flat: threeFlatValue
three.structured: threeStructuredValue

Activity A1: "Start"


Activity A3: "Log Props"

Attributes

  • Rule: import com.centurylink.mdw.config.PropertyManager println 'one: ' + PropertyManager.getProperty("one.prop") println 'two: ' + runtimeContext.getProperty("two.prop") println 'three.flat: ' + PropertyManager.getProperty("three.flat.config") println 'three.structured: ' + runtimeContext.getProperty("three.structured.config")
  • SCRIPT: Groovy

Activity A2: "Stop"
